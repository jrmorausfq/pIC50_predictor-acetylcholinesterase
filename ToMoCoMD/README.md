# ToMoCoMD

Place the following file in this folder:

```text
ToMoCoMD-CARDD_CLI.jar
```

ToMoCoMD-CARDD must be downloaded from the official website:

https://tomocomd.com/

The script expects this exact path:

```text
ToMoCoMD/ToMoCoMD-CARDD_CLI.jar
```

Third-party `.jar` files should not be uploaded to GitHub unless the corresponding license allows redistribution.
